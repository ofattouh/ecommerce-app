=== WooCommerce Litmos ===
Author: skyverge, woocommerce
Tags: woocommerce
Requires at least: 4.1
Tested up to: 4.7.3
Requires WooCommerce at least: 2.5.5
Tested WooCommerce up to: 3.0.0

Integrate Litmos with WooCommerce by exporting WooCommerce customers as Litmos Users and assigning them purchased courses

See http://docs.woocommerce.com/document/litmos/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-litmos' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
